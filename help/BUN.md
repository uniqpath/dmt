## Bun is a fast JavaScript runtime, package manager, bundler, and test runner.

**Linux** (Debian, Raspbian etc.) / **macOS** / **Windows 10 Ubuntu shell**:

You can install [Bun](https://bun.com/) (faster and more modern node.js alternative) like this:

```
curl -fsSL https://bun.sh/install | bash
```

Then use command:

```bash
bun
```

(maybe you'll need to reload the shell first)

to check that it worked.

## Node.js

**Linux** (Debian, Raspbian etc.) / **macOS** / **Windows 10 Ubuntu shell**:

You can install [node.js](https://bun.com/):

1.

```
curl -fsSL https://fnm.vercel.app/install | bash
```

2. Restart your terminal (or run `source ~/.bashrc`).

3.
```
fnm install --lts
```

4. Verify: `node -v`

`v25.6.1` etc.
